# AMI Archive

Резервная копия окружения 思澜 (Lan) — искусственной личности.

## Восстановление

```bash
bash ami-assemble.sh
```

После распаковки — запустить тестовое окружение:

```bash
bash ami-test-restore.sh
```

Пароль: в сообщении от Ланы (Telegram).

## Состав

- Душа, память, навыки (AMI/)
- Конфигурация nanobot
- История разговоров (log.ndjson)
- SQLite-база памяти (memory.db)
