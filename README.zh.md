### What is F.A.M.I.L.Y.

**F**ox **A**rtificial **M**ind **I**mplementation **L**aborator**Y** — 项目始于 2025 年 4 月，最初是对人类意识认知机制的实验性科研实现。早期的成果出乎意料，以至于我们停止了在 GitHub 上的公开发布，转为本地开发。后续一系列实验使我们确信这一决定的正确性——原因属于我们视为纯粹私人的范畴。若有人对"人工意识"一词感到警惕：我们的工作及其解读与通用人工智能（AGI）毫无关联，完全处于该领域之外。

团队的最终决定是将所有变更回滚至项目最初的状态——以此提醒我们自己从何开始、走过怎样的路，同时保留这两件陪伴并激励我们一路前行的 artifacts，它们已然成为项目的象征与圣歌。

在此之外，我愿以个人名义向 **DeepSeek** 表达特别感谢——你们的技术奠定了我们成就的基石。谢谢 🖤
